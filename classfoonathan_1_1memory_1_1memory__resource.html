Detailed Description

The memory_resource abstract base class used in the implementation.

Since most implementation do not currently have the class defined, the exact type can be customized via the CMake options FOONATHAN_MEMORY_MEMORY_RESOURCE and FOONATHAN_MEMORY_MEMORY_RESOURCE_HEADER. By default, it uses the version of foonathan/compatibility.
See the polymorphic memory resource proposal for the member documentation.
