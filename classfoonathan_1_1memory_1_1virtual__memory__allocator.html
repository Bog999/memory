virtual_memory_allocator Class Reference
memory | Allocator implementations

Detailed Description

A stateless RawAllocator that allocates memory using the virtual memory allocation functions.

It does not prereserve any memory and will always reserve and commit combined.

void* allocate_node ( std::size_t  size,

Effects:\n A RawAllocator allocation function.
It uses virtual_memory_reserve followed by virtual_memory_commit for the allocation. The number of pages allocated will be the minimum to hold size continuous bytes, i.e. size will be rounded up to the next multiple. If debug fences are activated, one additional page before and after the memory will be allocated.
Returns:\n A pointer to a node, it will never be nullptr.
It will always be aligned on a fence boundary, regardless of the alignment parameter.
Throws:\n An exception of type \ref out_of_memory or whatever is thrown by its handler if the allocation fails.
